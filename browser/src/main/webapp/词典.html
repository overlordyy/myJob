VNS介绍

  

VNS是AMT社区在经历多年的理念沉淀后，充分融合了AMT价值理念的第一个生态平台项目。AMT社区相信区块链技术必将改变人类生活方式，未来社会的联通将会在现有的信息互联的基础上进化成价值互联，在区块链构建的信任体系中，各式各样的人或物将自我价值通过区块链网络进行传递，形成丰富的价值互联网络生态，这最终将极大地提升社会生产效率。

  

VNS公链所采用的技术包含但不限于以下技术：

-  智能合约级别零知识证明功能

-  基于环形签名的匿名交易(Confidential Transaction)

-  CryptoNight挖矿算法



VNS挖矿奖励

  

VNS挖矿奖励分为4大类，分别对应的奖励机制和金额如下：

- 预挖：3,860,977,695

- 普通奖励：209

- 叔块奖励 ：( 叔块高度 + 8 - 包含叔块的区块的高度 ) * 普通区块奖励 / 8

   （注：叔块奖励=6/8或者7/8的普通奖励，也就是157个或者183个）

- 叔块引用奖励:   1/32*普通奖励*uncle_count  (0 ≤ uncle_count ≤ 2)
